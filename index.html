BackInTime

Requisiti minimi: macOS 13 Ventura • Apple Silicon

Utility per macOS che ripristina la data di creazione mostrata dal Finder per foto e video, riportandola alla reale data di scatto quando il file contiene i metadati EXIF o equivalenti.

Cosa fa

Quando una foto o un video vengono modificati ed esportati, la data di creazione mostrata dal Finder può diventare quella dell'esportazione invece di quella originale di scatto.

BackInTime legge i metadati di creazione presenti nel file e aggiorna la data di creazione visibile nel Finder, così da renderla nuovamente coerente con la data reale del contenuto.

Come funziona

Basta trascinare nell'app una o più foto o video contenenti metadati di creazione. BackInTime legge la data originale salvata nel file e la applica alla data di creazione mostrata dal Finder.

L'app è pensata per i casi in cui un file esportato da Foto o da altri editor mantenga correttamente i metadati di scatto, ma perda la corrispondenza con la data di creazione mostrata nel file system.

L'app è sviluppata in Swift e non utilizza librerie esterne.
